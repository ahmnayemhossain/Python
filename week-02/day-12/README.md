# Day 12 - Password Generator V1

## What I Learned

- How `for` loops repeat steps
- How `range()` controls loop count
- How an accumulator builds a result

## Features

- Lets the user choose letter, symbol, and number counts
- Builds a random password
- Shuffles characters for better output

## How to Run

```bash
python main.py
```

## Example Output

```text
Welcome to Password Generator V1!
How many letters would you like? 4
How many symbols would you like? 2
How many numbers would you like? 2
Generated password: aB3@q!7Z
```

## Mistakes I Fixed

- Built the password step by step with loops
- Shuffled the result to avoid grouped character types
