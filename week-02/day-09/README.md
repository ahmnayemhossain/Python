# Day 09 - Coin Toss and Dice Simulator

## What I Learned

- How to import the `random` module
- How `random.choice()` works
- How `random.randint()` works

- Simulates a coin toss
- Simulates a dice roll
- Prints random results in the terminal

## How to Run

python main.py

Coin toss result: Heads
