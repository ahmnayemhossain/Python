# Day 08 - Treasure Island Text Game

## What I Learned

- How nested `if` blocks work
- How `elif` and logical flow shape a text game
- How lowercase input helps avoid mismatch

## Features

- Uses multiple decisions from the user
- Has winning and losing branches
- Keeps the game fully terminal-based

## How to Run

```bash
python main.py
```

## Example Output

```text
Welcome to Treasure Island!
Your mission is to find the treasure.
Left or Right? left
Swim or Wait? wait
Which door? Red, Blue, or Yellow? yellow
You found the treasure! You win!
```

## Mistakes I Fixed

- Normalized input with lowercase text
- Ordered decisions step by step to avoid confusion

## Future Improvement

- Add more rooms and alternate endings
