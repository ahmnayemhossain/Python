# Day 10 - Who Pays the Bill App

## What I Learned

- How to create and update lists
- How to split user input into a list
- How to choose a random item from a list

## Features

- Accepts comma-separated names
- Builds a clean list of participants
- Chooses one person randomly

## How to Run

```bash
python main.py
```

## Example Output

```text
Welcome to the Who Pays the Bill App!
Enter names separated by commas: Rahim, Karim, Jui
Karim will pay the bill today.
```

## Mistakes I Fixed

- Trimmed extra spaces from names
- Filtered out empty list items
