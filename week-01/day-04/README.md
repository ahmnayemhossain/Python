# Day 04 - BMI Interpretation Calculator

## What I Learned

- How to use math operators in Python
- How to round numbers for cleaner output
- How to apply BMI logic with conditions

## Features

- Takes height and weight from the user
- Calculates BMI using the standard formula
- Shows a BMI category based on the result

## How to Run

```bash
python main.py
```

## Example Output

```text
Welcome to the BMI Interpretation Calculator!
Enter your height in meters: 1.75
Enter your weight in kilograms: 68
Your BMI is 22.2, which means you are normal weight.
```

## Mistakes I Fixed

- Rounded the BMI to keep the output readable
- Used clear category ranges for the BMI result
