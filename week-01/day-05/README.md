# Day 05 - Age-Based Ticket Price App

## What I Learned

- How to use `if`, `elif`, and `else`
- How comparison operators control program flow
- How to map age ranges to output values

## Features

- Takes age input from the user
- Checks which ticket bracket matches
- Prints the final ticket price

## How to Run

```bash
python main.py
```

## Example Output

```text
Welcome to the Age-Based Ticket Price App!
Enter your age: 16
Your ticket price is: 350 BDT
```

## Mistakes I Fixed

- Ordered the conditions from smaller range to larger range
- Kept the output simple so the result is easy to read
