# Day 02 - Band or Brand Name Generator

## What I Learned

- How to collect user input with `input()`
- How to store values in variables
- How to combine strings into one result

## Features

- Takes city name from the user
- Takes favorite word or pet name from the user
- Generates a combined band or brand name

## How to Run

```bash
python main.py
```

## Example Output

```text
Welcome to the Band or Brand Name Generator!
What is the name of your city? Dhaka
What is your favorite word or pet name? Thunder
Your new name could be: Dhaka Thunder
```
