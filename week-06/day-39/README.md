# Day 39 - Word Frequency Counter

## What I Learned

- How dictionary comprehensions build mappings quickly
- How sets help find unique values first
- How text can be summarized into counts

## Features

- Takes a sentence from the user
- Counts each unique word
- Prints frequency results in sorted order

## How to Run

```bash
python main.py
```

## Example Output

```text
Welcome to Word Frequency Counter!
Enter a sentence: python code python practice
code: 1
practice: 1
python: 2
```

## Mistakes I Fixed

- Lowercased text before counting words
- Counted only unique words in the final dictionary

## Future Improvement

- Remove punctuation before counting
