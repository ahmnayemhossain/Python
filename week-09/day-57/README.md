# Day 57 - Birthday Email Draft Generator

## What I Learned

- How email drafts are structured
- What SMTP is used for
- How automation can prepare messages safely before sending

## Features

- Generates a birthday email subject
- Generates a clean email body draft
- Includes the current draft date

## How to Run

```bash
python main.py
```

## Example Output

```text
Welcome to Birthday Email Draft Generator!
Enter friend's name: Jui
Enter your name: Nayem

Subject:
Happy Birthday, Jui!
```

## Mistakes I Fixed

- Kept this tool as a draft generator instead of sending real email
- Structured the subject and body separately
