# Day 43 - Coffee Machine OOP V1

## What I Learned

- How classes define blueprints
- How objects store their own state
- How methods change object data

## Features

- Creates a coffee machine object
- Tracks water, milk, and coffee resources
- Makes one latte if resources are available

## How to Run

```bash
python main.py
```

## Example Output

```text
Welcome to Coffee Machine OOP V1!
Water: 500ml, Milk: 400ml, Coffee: 100g
Latte is ready!
Water: 300ml, Milk: 250ml, Coffee: 76g
```

## Mistakes I Fixed

- Stored machine values as object attributes
- Reduced resources only when enough ingredients existed
