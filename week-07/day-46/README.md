# Day 46 - Employee and Manager Payroll Model

## What I Learned

- How inheritance connects parent and child classes
- How child classes can extend base behavior
- How method overriding changes results

## Features

- Models an employee and a manager
- Uses inheritance to share name and salary
- Adds a manager bonus through an overridden method

## How to Run

```bash
python main.py
```
