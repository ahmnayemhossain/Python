# Day 16 - Calculator Functions

## What I Learned

- How to define functions
- How parameters pass values into functions
- How return values make functions reusable

## Features

- Supports add, subtract, multiply, and divide
- Uses separate functions for each operation
- Prints the result based on user choice

## How to Run

```bash
python main.py
```

## Example Output

```text
Welcome to Calculator Functions!
Enter first number: 12
Choose operation (+, -, *, /): *
Enter second number: 5
Result: 60.0
```

## Mistakes I Fixed

- Separated each operation into its own function
- Added a safe divide-by-zero check
