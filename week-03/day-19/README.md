# Day 19 - Student Score Analyzer

## What I Learned

- How list methods update list values
- How sorting changes data order
- How `min()` and `max()` help analyze scores

## Features

- Accepts multiple scores from the user
- Prints sorted, lowest, highest, and average values
- Uses built-in list tools for analysis

## How to Run

```bash
python main.py
```

## Example Output

```text
Welcome to the Student Score Analyzer!
Enter student scores separated by spaces: 78 92 65 88
Sorted scores: [65, 78, 88, 92]
Lowest score: 65
Highest score: 92
Average score: 80.75
```
