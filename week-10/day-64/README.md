# Day 64 - Mile to Kilometer Converter GUI

## What I Learned

- How to create a Tkinter window
- How labels, buttons, and entry fields work together
- How GUI input connects to a conversion function

## Features

- Takes miles as input
- Converts the value to kilometers
- Updates the result on the window

## How to Run

```bash
python main.py
```

## Example Output

```text
Opens a small GUI window for mile-to-kilometer conversion.
```
