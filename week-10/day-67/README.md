# Day 67 - Password Manager GUI V1

## What I Learned

- How Tkinter works with JSON storage
- How form fields can collect password data
- How GUI actions can trigger file saving

## Features

- Collects website, email, and password
- Saves records into a JSON file
- Shows a messagebox after saving

## How to Run

```bash
python main.py
```

## Example Output

```text
Opens a password manager GUI that saves data to JSON.
```

## Mistakes I Fixed

- Loaded existing JSON before writing new data
- Kept form fields separate for cleaner storage
