# Day 65 - Simple Calculator GUI

## What I Learned

- How `grid` and `pack` differ in layout design
- How button presses can update a shared display
- How a simple calculator can be built with GUI widgets

## Features

- Uses a title label with `pack`
- Uses `grid` for the calculator buttons
- Evaluates a typed math expression

## How to Run

```bash
python main.py
```

## Example Output

```text
Opens a calculator window with buttons and a display.
```

## Mistakes I Fixed

- Kept layout organized by mixing `pack` and `grid` safely in different containers
- Added a clear button for quick reset
