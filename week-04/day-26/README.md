# Day 26 - Random Quote Generator

## What I Learned

- How to import built-in modules
- How to import values from another Python file
- How a small module keeps data organized

## Features

- Stores quotes in a separate module
- Chooses one quote randomly
- Prints a simple motivational line in the terminal

## How to Run

```bash
python main.py
```

## Example Output

```text
Welcome to the Random Quote Generator!
Discipline beats motivation.
```

## Mistakes I Fixed

- Imported quotes from a separate file instead of hardcoding everything in `main.py`
- Used `random.choice()` on the quote list

## Future Improvement
