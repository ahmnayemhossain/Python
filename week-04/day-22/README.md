# Day 22 - Contact Book V1

## What I Learned

- How dictionaries store key-value pairs
- How to add, update, and search dictionary data
- How menu-based dictionary apps work

## Features

- Adds contacts with name and phone number
- Lists all saved contacts
- Searches a contact by name

## How to Run

```bash
python main.py
```

## Example Output

```text
Welcome to Contact Book V1!

1. Add contact
2. View all contacts
3. Search contact
4. Exit
Choose an option: 1
Enter contact name: Rahim
Enter phone number: 01700000000
Contact saved.
```
